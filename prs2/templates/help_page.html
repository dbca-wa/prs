PRS locations in QGIS

PRS locations are available as a GIS data source (WMS/WFS) via the Kaartdijin Boodja system.

  1. In QGIS, create a new WMS or WFS connection.
  2. Name the connection KB and use {{ geoserver_url }}/ows as the connection URL.
  3. Visit https://auth2.dbca.wa.gov.au and generate an access token to authenticate yourself with KB.
  4. In QGIS, configure the new WMS connection to create an authentication configuration:
    • Name: KB auth
    • Type: Basic authentication
    • Username: your email address
    • Password: the access token value created earlier
  5. In the new WMS connection, add the prs_locations layer to your QGIS project.
